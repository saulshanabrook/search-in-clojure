log-wrap

(log-wrap g)

Inputs: [g :- utils/Graph] Returns: utils/Graph

Modify graph spec g, producing a new graph spec.

If any node produces a function, it wraps that function to log when it is called.

This is useful when debugging where your application is spending time.
