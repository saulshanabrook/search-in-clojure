search.cli

-main

(-main recorder-path config-path)

Creates and executes a run. It takes two required positional arguments: * recorder-path: Path to the recorder (i.e. search.recorders.text/timbre) * config-path: Path to the config var (i.e. search.examples.list/config)

If you set the VALIDATE_SCHEMA env variable it will validate the schemas as well.
